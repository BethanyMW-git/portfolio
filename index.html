PROJECTS

Check out some of the work that I accomplished during the Digital Crafts Web Development Bootcamp. My portfolio includes examples of front-end, back-end, React, and full-stack applications.

iPrep

iPrep is a Full-stack React project that served as my Digital Crafts Capstone Project. iPrep is a Meal Kit Service App that allows for user authentication for Login and Sign Up features. From the Homepage, users are prompted to take a Quiz to customize their order, after which they have access to Checkout. Users can also access a free searchable/sortable Catalog that provides free recipes from the Spoonacular API.

Lofi Live Music App

Lofi Live is a music app built as a front-end project. Users can select songs from a navigation panel. On click, the selected video will play. User clicks are counted and displayed. The selected songs are streamed in from live 24/7 lofi music stations on YouTube.

MangaBear E-Commerce App

MangaBear is a full-stack, e-commerce app. The app allows for user authentication for Login and Sign Up features. After logging in, users gain access to a catalog of manga titles for purchase.

Weather App

This weather app is a front-end project in which users can search their location to gain accurate weather data pulled from the OpenWeather API displayed in different categories within a responsive dashboard of semi-transparent tiles. The background dynamically changes between weather appropriate looping videos according to data found at that location.
